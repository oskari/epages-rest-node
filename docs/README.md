# API reference (ePages)

API reference content from [ePages Developer Documentation](https://github.com/ePages-de/epages-docs), trimmed to what is relevant for implementing and extending this REST SDK.

## Layout

| Path | Contents |
|------|----------|
| **rest/** | REST API markdown: [overview](rest/overview.md), data types, headers, rate limits, versions |
| **raml/** | REST API RAML specs, JSON schemas, and examples (products, orders, customers, carts, categories, etc.) |
